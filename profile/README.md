**Bitropy is the enterprise LLM / MCP gateway and AI governance platform — Kubernetes-native, deployable on-premises, in VPC, or air-gapped — for regulated companies adopting LLMs and AI agents under EU AI Act, GDPR, DORA, and SOC 2.**

We sit between AI systems (LLMs, agents, copilots) and enterprise data and tools, providing a single secure control layer where authentication, authorization, observability, policy enforcement, and cost management are handled by design — not bolted on per integration.

## The Platform

| Component | What it does |
|---|---|
| **LLM Gateway** | Unified, OpenAI-compatible API across OpenAI, Anthropic, Mistral, and self-hosted models. Semantic routing, load balancing, failover. |
| **MCP Gateway** | Secure, auditable AI-to-data connectivity over the Model Context Protocol. Centralized auth, isolation, and aggregation across MCP servers. |
| **MCP Registry** | Source of truth for available MCP servers — versioning, approvals, ownership, self-service catalog. |
| **MCP Runner** | Isolated runtime for MCP servers with secrets handling, network policy, and per-server resource limits. |
| **Policy Engine** | OPA / Kyverno-based runtime guardrails: PII redaction, model allow-listing, data residency, prompt-injection patterns, budget enforcement. |
| **Observability** | OpenTelemetry tracing, structured audit logs, Prometheus metrics, Grafana dashboards. Every prompt, tool call, and response is traceable. |
| **FinOps** | Token metering, per-team budgets, anomaly detection, cost attribution. |

## Built for every team involved

- **Security & CISOs** — central auth, PII masking, isolation, a single audit trail.
- **Platform & DevOps** — registry, GitOps deploys, rate limits, a standardized runtime.
- **Developers & AI Engineers** — one gateway, self-service catalog, dev/prod parity.
- **Compliance & Legal** — clear policy enforcement point, mappings to SOC 2, ISO 27001, ISO/IEC 42001.

## Deployment

- **On-premises** into your existing Kubernetes cluster
- **Customer VPC** (AWS, Azure, GCP, VMware, bare metal) — Bitropy-managed
- **Air-gapped** for defense, critical infrastructure, and high-security environments

Full data sovereignty. Open standards. No vendor lock-in.
